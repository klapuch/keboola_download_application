# keboola_download_application

It downloads remote authenticated files.

Usage
-----

`Keboola config file`

```
{
    "url": "http://remote.file.cz",
    "username": "harry",
    "password": "secret",
    "old_delimiter": ";",
    "new_delimiter": ","
}
```

`old_delimiter` and `new_delimiter` are optional. They are `,`(comma) by default.
